SafeWalk

Technical Highlights

  • Object detection powered by YOLOv5s, fine-tuned on 28 object classes including cane, scooter, bollard, stairs, etc.
  • Training dataset: ~15GB, 91,000+ files. Final model metrics: Precision 0.63, Recall 0.51, mAP@0.5: 0.51.
  • Key challenges solved: class imbalance (handled via augmentation and undersampling), catastrophic forgetting (solved via replay-based learning).
  • Scene understanding and narration via Qwen2 Visual Language Model. Compared to Paligemma, Qwen2 provided richer descriptions and performed up to 67% better in navigation-relevant categories like spatial orientation and environmental awareness.
  • YOLOv5s chosen over Faster-RCNN for deployment efficiency, latency, and compatibility with mobile environments.

Models were optimized for mobile deployment, with future extensibility to smart glasses or cane-mounted camera platforms. Real-time latency was a key driver of architecture and model selection.

SafeWalk

SafeWalk Demo

Below is a live demonstration of the SafeWalk mobile assistant app in action. This video walkthrough showcases how SafeWalk supports visually impaired users through real-time obstacle detection, adaptive voice alerts, and scene description.

Demonstrated Features

  • Obstacle Detection: Identifies nearby hazards like scooters, potholes, and poles using a fine-tuned YOLOv5s model.
  • Voice Alerts: Real-time audio notifications adjust based on object proximity and type.
  • Scene Description: Qwen2 visual language model provides rich narration to help users build spatial awareness.
  • Multi-modal Interaction: Supports spoken and gesture-based controls for accessibility.
  • Front/Back Camera Toggle: Users can switch views depending on device orientation or context.

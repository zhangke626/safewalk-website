SafeWalk

Empowering Visually Impaired Navigation

SafeWalk is a mobile assistant app designed to help visually impaired individuals safely navigate urban environments using real-time object detection and scene description powered by computer vision and AI.

  • Learn More

Key Features

Obstacle Detection

Real-time detection of obstacles using a fine-tuned YOLOv5s model with 28 custom classes.

Adaptive Voice Alerts

Proximity-based audio feedback that informs users of nearby dangers as they move.

Scene Description

On-demand visual scene narration via Qwen2 Visual Language Model.

Accessibility-First Design

Supports Speech to Text, Text to Speech, and Multiple Languages.

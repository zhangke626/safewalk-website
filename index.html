Empowering Visually Impaired Navigation

SafeWalk empowers visually impaired individuals with the safety, independence, and confidence to navigate the world. Through real-time object detection and rich scene descriptions, our mobile assistant app delivers smart audio alerts and situational awareness tailored to urban environments.

Key Features

Obstacle Detection

Real-time detection of obstacles using a fine-tuned YOLOv5s model with 28 custom classes including potholes, scooters, and poles.

Adaptive Voice Alerts

Proximity-based audio feedback to help users safely react to nearby hazards.

Scene Description

On-demand narration of surroundings powered by the Qwen2 Visual Language Model, optimized for spatial orientation and environmental awareness.

Accessibility-First Design

Supports gesture and spoken feedback, integrates with screen readers, and offers multilingual support for inclusive use.
